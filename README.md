## Requirements

Make sure that your server is configured with following PHP version and extensions:

- PHP 8.1+

## Installation

You can install the package via composer:

```bash
composer require roadrunner-php/version-checker
```

## Testing

```bash
composer test
```

```bash
composer psalm
```

```bash
composer cs
```
